# 🐉 Game of Thrones — Видеоплеер

Веб-приложение на Angular 19 для просмотра всех сезонов и серий сериала «Игра Престолов». Поддерживает SSR, сохранение прогресса просмотра и имитацию рекламы перед началом эпизода.

---

## Стек технологий

- **Angular 19** — standalone-компоненты, Angular SSR (Server-Side Rendering)
- **TypeScript 5.7**
- **CSS** — адаптивная вёрстка с медиазапросами
- **json-server** — REST API на основе `db.json` (порт 3000)
- **Express** — SSR-сервер
- **RxJS** — реактивные HTTP-запросы
- **concurrently** — одновременный запуск Angular и json-server

---

## Возможности

- **Главная страница** — hero-блок с описанием сериала и кнопкой «Смотреть», галерея всех 8 сезонов с обложками
- **Страница всех серий** — сетка эпизодов по каждому сезону; быстрый переход к любой серии по форме выбора «сезон + эпизод»
- **Видеоплеер** — нативный HTML5-плеер с постером, навигация по сезонам (◄ / ►) и список серий внутри сезона
- **Реклама перед видео** — 10-секундный YouTube-ролик в iframe с кнопкой «Пропустить» (появляется через 10 сек)
- **Сохранение прогресса** — текущий сезон, серия и время воспроизведения сохраняются в `localStorage` и восстанавливаются при следующем визите
- **Адаптивный хедер** — бургер-меню на экранах ≤992px
- **Полная адаптивность** — корректное отображение на мобильных, планшетах и десктопах

## Структура проекта

```
src/app/
├── component/
│   ├── header/          # Навигационная шапка с бургер-меню
│   ├── footer/          # Футер с контактами и ссылками
│   ├── main-cont/       # Hero-блок главной страницы
│   ├── season-cont/     # Галерея сезонов (главная)
│   ├── season-see/      # Сетка серий одного сезона
│   └── episode-form/    # Форма быстрого выбора сезона и серии
├── pages/
│   ├── index/           # Главная страница
│   ├── see-all/         # Страница всех серий
│   └── watch-episode/   # Страница плеера
└── data/
    ├── models/
    │   └── season.model.ts  # Интерфейсы Season и Episode
    ├── Serv/
    │   └── season.service.ts # HTTP-сервис для загрузки сезонов
    └── Dbs/db.json           # База данных json-server
```

## Маршруты

| Путь | Компонент | Описание |
|---|---|---|
| `/` | → redirect | Перенаправляет на `/index` |
| `/index` | `IndexComponent` | Главная: hero + галерея сезонов |
| `/SeeAll` | `SeeAllComponent` | Все серии по сезонам + форма выбора |
| `/watch-episode` | `WatchEpisodeComponent` | Видеоплеер |

## Установка и запуск

### Требования

- Node.js v18+
- npm v9+

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

Запускает Angular dev-server и json-server одновременно:

```bash
npm start
```

- Приложение: [http://localhost:4200](http://localhost:4200)
- API: [http://localhost:3000](http://localhost:3000)

## API (json-server)

Сервер поднимается на основе `src/app/data/Dbs/db.json`.

| Метод | Endpoint | Описание |
|---|---|---|
| `GET` | `/seasons` | Получить все сезоны с эпизодами |
| `GET` | `/seasons?number=1` | Получить конкретный сезон |

## Заметки о реализации

- Прогресс воспроизведения сохраняется при переходе между страницами (`NavigationStart`) и закрытии вкладки (`beforeunload`)
- SSR настроен через `@angular/ssr` с режимом `Prerender` для всех маршрутов
- Рекламный блок отображает YouTube-ролик в `<iframe>` с `DomSanitizer.bypassSecurityTrustResourceUrl`; кнопка «Пропустить» появляется через 10 секунд
- `SeasonService` обращается к `http://localhost:3000/seasons` — при деплое без json-server URL нужно заменить на реальный бэкенд
